Fundamentos de HTML

¿Que es HTML?


HTML es la sigla de HyperText Markup Language, que en español significa Lenguaje de Marcado de Hipertexto. Es el lenguaje estándar utilizado para la creación y estructuración de páginas web. HTML permite organizar y mostrar el contenido mediante etiquetas o tags, las cuales indican al navegador cómo debe mostrarse la información (títulos, párrafos, imágenes, formularios, etc.). Los navegadores interpretan este código y lo convierten en la página visual que ves en internet.

¿Que significa HTML?


H

HyperText


Capacidad de conectar documentos mediante hipervinculos (links), permitiendo navegar entre diferentes paginas.

M

Markup


Uso de etiquetas para marcar y estructurar el contenido dentro de un documento.

L

Language


Conjunto de reglas y sintaxis utilizadas para escribir y organizar el contenido web.


¿Para que sirve HTML?


  • Definir la estructura básica y jerarquía.
  • Crear títulos y párrafos de texto.
  • Insertar imágenes y contenido multimedia (videos/audios).
  • Crear enlaces y organizar datos en tablas o listas.
  • HTML trabaja junto a:
  • CSS: Estilo y diseño.
  • JavaScript: Interactividad y dinámica.

✅ Ventajas

  • Fácil de aprender: Sintaxis clara para principiantes.
  • Compatibilidad: Funciona en todos los navegadores modernos.
  • Gratuito y Abierto: Sin costo de licencias.
  • Base Universal: Todos los sitios web lo utilizan.
  • Integración: Se combina con CSS y JS.

❌ Desventajas

  • No es programación: No ejecuta lógica compleja por sí solo.
  • Páginas Estáticas: Sin interactividad sin JavaScript.
  • Visualmente Pobre: Necesita CSS para verse bien.
  • Limitación de Datos: No se conecta directo a bases de datos.

Conclusión


HTML es el corazón de la web. Aunque tiene limitaciones, su facilidad de uso y capacidad de integración lo convierten en una herramienta esencial para cualquier desarrollador web profesional.

Etiquetas en HTML

¿Que son las etiquetas?


Las etiquetas HTML son elementos utilizados para definir y estructurar el contenido de una página web. Indican al navegador cómo debe interpretarse y mostrarse cada parte del contenido (textos, imágenes, tablas, etc.).

Están formadas por palabras clave encerradas entre signos de menor y mayor que < >. Generalmente se componen de una apertura y un cierre.

<p> Este es un párrafo </p>

¿Para que sirven?


1

Organización


Permiten que los navegadores interpreten el contenido correctamente (títulos, listas, videos).

2

Jerarquía


Definen qué información es más importante mediante encabezados y secciones.

3

Interactividad


Facilitan la creación de formularios y enlaces para conectar con el usuario.

¿Como se usan?


  • Se escriben directamente en el documento. La mayoría sigue la estructura básica: <etiqueta>contenido</etiqueta>.
  • <h1>Título principal</h1>
  • Además, pueden incluir atributos (propiedades adicionales):
  • <img src="imagen.jpg" alt="Descripción" width="200">
  • <p>Este es un párrafo de texto.</p>
  • src: ruta del archivo.
  • alt: descripción para accesibilidad.
  • width: define el tamaño.

✅ Ventajas

  • Fácil de aprender: Nombres simples y sintaxis lógica.
  • Organización: Estructura clara y ordenada.
  • Compatibilidad: Funcionan en todos los navegadores.
  • Base Universal: Estándar mundial de la web.
  • Integración: Se combinan perfecto con CSS y JS.

❌ Desventajas

  • Sin lógica: No realizan cálculos ni procesos complejos.
  • Dependencia: Requieren CSS para diseño avanzado.
  • Mantenimiento: El código puede ser extenso en sitios grandes.
  • Datos: No se conectan directo a bases de datos.

Las etiquetas son los ladrillos de la web. Sin ellas, el navegador no sabría diferenciar entre un simple texto y un botón interactivo. Dominarlas es el primer paso para ser un desarrollador profesional.
