4. Etiquetas de Formularios

Elementos para interactuar con el usuario y recopilar datos de entrada.

Etiqueta: form

Definicion

Contenedor principal que agrupa campos de entrada para enviar informacion al servidor.

Sintaxis basica / demostracion visual


Demo interactivo — Crear cuenta

Uso Adecuado

Se usa para crear secciones de contacto, registros o inicios de sesion.

Buenas practicas:
  • Incluir etiquetas descriptivas para cada campo.
  • Validar que los datos sean correctos antes de enviar.
Etiqueta: input

Definicion

Campo versatil donde el usuario introduce datos. No necesita etiqueta de cierre.

Sintaxis basica / demostracion visual

Uso Adecuado

Segun su atributo type, puede ser texto, correo, contrasena, numero o fecha.

Buenas practicas:
  • Usar el tipo correcto (email para correos, number para numeros).
  • Usar el atributo placeholder para orientar al usuario.
Etiqueta: label

Definicion

Define un texto asociado a un campo de entrada.

Sintaxis basica / demostracion visual

Uso Adecuado

Mejora la accesibilidad, ya que al hacer clic en el texto se activa el input asociado.

Etiqueta: button

Definicion

Crea un boton interactivo para ejecutar acciones o enviar formularios.

Sintaxis basica / demostracion visual

Uso Adecuado

Se usa para enviar formularios o ejecutar acciones con JavaScript.

Buenas practicas:
  • Usar textos claros como "Enviar" o "Registrarse".
  • Definir el tipo de boton (submit o button).
