HTML es el lenguaje utilizado para crear páginas web. En este blog aprenderás algunas etiquetas importantes y cómo funcionan dentro de un sitio web.

Categoría: Etiquetas de Estructura

<header>

Define el encabezado de una página o sección.

<nav>

Se usa para crear menús de navegación.

<footer>

Representa el pie de página del sitio web.

Categoría: Etiquetas de Texto

<h1>

Se utiliza para títulos principales.

<p>

Se usa para escribir párrafos de texto.

<a>

Permite crear enlaces a otras páginas o sitios web.

Categoría: Etiquetas Multimedia

<img>

Permite mostrar imágenes dentro de la página web.

<video>

Permite reproducir videos dentro de una página.

<audio>

Permite agregar audio en una página web.
